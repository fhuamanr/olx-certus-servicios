## Estructura
1. Desarrolla la capa de persistencia a datos utilizando Spring data JPA.
2. Implementa la configuración de Spring data JPA.
3. Ejecuta las anotaciones utilizando Spring Data JPA.
4. Implementa las relaciones de entidades utilizando anotaciones de Spring Data JPA.
5. Implementa una interfaz utilizando CrudRepository y/o JPARepository.
